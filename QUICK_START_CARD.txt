┌───────────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING:                                                     │
└───────────────────────────────────────────────────────────────────────┘

    ❌  "Ollama not found"
        → Install: https://ollama.com/download
        → Restart computer
        
    ❌  "Port already in use"
        → Run STOP_ALL first
        → Wait 5 seconds
        → Run START_ALL again
        
    ❌  "PowerShell scripts disabled"
        → Run PowerShell as Admin
        → Run: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
        → Try again
        
    ❌  Frontend won't load
        → Wait 30 seconds after startup
        → Check "Nexus - Frontend UI" window for errors
        → Make sure you ran "npm install" in src/frontend
        
    ❌  Model download stuck
        → Check your internet connection
        → llama3.1:8b is 4.7GB (takes 5-15 min first time)

┌───────────────────────────────────────────────────────────────────────┐
│  FIRST-TIME SETUP (One Time Only):                                   │
└───────────────────────────────────────────────────────────────────────┘

    1. Install Ollama:       https://ollama.com/download
    2. Install Python 3.11+: https://python.org
    3. Install Node.js 18+:  https://nodejs.org
    
    4. Install dependencies:
       cd path\to\project
       pip install -r requirements.txt
       cd src\frontend
       npm install
       
    5. Run START_ALL.ps1 → Will auto-pull LLM models (15 min first time)
