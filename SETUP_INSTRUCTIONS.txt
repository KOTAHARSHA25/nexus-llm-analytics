STEP 1: Extract the Package
----------------------------
- Unzip the nexus-llm-analytics-dist folder
- Place it somewhere convenient (e.g., C:\Projects\)

STEP 2: Setup Python Environment
---------------------------------
Open Command Prompt/PowerShell in the project folder:

   cd nexus-llm-analytics-dist
   python -m venv env
   
   # Windows:
   env\Scripts\activate
   
   # Linux/Mac:
   source env/bin/activate
   
   pip install -r requirements.txt

STEP 3: Setup Frontend
-----------------------
   cd src\frontend
   npm install
   cd ..\..

STEP 4: Install AI Models
--------------------------
Make sure Ollama is running, then:

   ollama pull llama3.1:8b
   ollama pull phi3:latest

(This will download ~4-6 GB of models - takes 5-15 minutes)
